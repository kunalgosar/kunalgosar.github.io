Scheme Interpreter

Built a scheme interpreter as part of a class project for CS 61A - Structure and Interpretation of Computer Programs

The Interpreter has functionality for quotation, lambda functions, conditional statements, local variable definitions, mu expressions (dynamic scope) and Tail Recursion.
